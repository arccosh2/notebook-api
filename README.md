# notebook-api

## 概要

Fast API を使用した TODO アプリ用の API。
ローカルでサーバーを立てるときは以下のコマンドを実行する。

```
uvicorn main:app --reload
```

ローカルで起動しているとき、URL の末尾に`/docs`を付与してアクセスすると Swagger UI を開くことができる。
